July 31 Information

In the last class we reviewed for and while loops in C++. The main idea was that while loops are easiest to use when you cannot effectively determine how many times you want to run a loop. On the other hand, for loops are easiest to use when you know exactly or can calculate how many times you want to run a loop. The main examples we covered were calculating the factorial of a number (for loop) and determining the 10,001st prime number (while loop).

We then started to cover how to write and use functions in C++ and how to transform already written code into a function. Specifically, a function to check divisibility, a function to compute the factorial, and a function to determine if a number is or is not prime. The key concepts are that a function returns a value for use in the main program and that they exist outside of the main function.

Finally, we began working with the Arduino we all purchased. We went over how the Arduino is a microcontroller and you can think of it like a very basic computer without a keyboard, mouse, or monitor and worked on a simple traffic light simulation with LEDs.

A reminder in case someone forgets, to compile a .cpp file, first you have to open a terminal/PowerShell window and use the 'cd' command to navigate to the folder containing your file; you may find the command 'ls' helpful to list everything in the current location. Then, for example a file called test.cpp, you compile it with: g++ test.cpp -o test. Finally, you can run it with .\test.exe.
